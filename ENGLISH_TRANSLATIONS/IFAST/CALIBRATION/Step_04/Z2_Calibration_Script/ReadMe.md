**05/11/2024**

**Generator for variable flow rate cube series with constant decreasing steps of 1% / 0.1% for calibration and bracketing of the Z2 (left nozzle) flow rate on the QIDI IFAST 3D printer.**

After producing the video [E01 of calibration tutorial step 04 — finding the flow rate % for a filament on Z2 QIDI IFAST](https://youtu.be/gn3A1lFX0vc) to complement [Step 04 — characterizing the flow rate % of a new filament](https://github.com/sudtek/IMPRIMANTES_3D/tree/f0b85f540401e21b22918730633c640832fa8458/QIDI/IFAST/CALIBRATION/Etape%2004), I realized that having to manually edit the flow rates of 11 cubes in IdeaMaker every time is slow, tedious, and error-prone. On top of that, the cube flow rate labels (e.g. 100%, 99%...) needed to be updated for each new cube series / step size to maintain any kind of consistency. Faced with this, I wrote a Python script called **Z2_generate.py** that automatically generates a calibration G-code for test cubes by asking the user 16 questions, producing 11 cubes labeled A (leftmost cube, highest flow rate) through K (rightmost cube, lowest flow rate). No more need to open the .IDEA source file in IdeaMaker and manually edit flow rates, labels, etc. Note that as of today (05/11/2024) this script only works for the left nozzle Z2 — a variant for calibrating Z1 is yet to be made.

It is printable and functional, but it is a special file — provided for reference only, as it allowed me to identify and differentiate the G-code blocks needed to build the template. This G-code is over 80,000 lines long. I intentionally assigned deliberately unusual and unique flow rate values to each element so I could distinguish them from one another, making sure no element I created had a flow rate of exactly 100%, allowing me to identify and locate each one. For example:

  Cube A flow rate → 101%, cube B → 99%, decreasing by -1% ... cube K → 90%.

  Purge tower → 105%.

  Legend A to K → 103%.

If you want to test a calibration G-code generated for the QIDI IFAST, you can download one of the two files above — *but note* that they were generated to print with PETG on Z2 (and PLA on Z1). If that matches your configuration you can run them directly on your IFAST, otherwise you will need to generate your own series by running [Z2_generate.py](https://github.com/sudtek/IMPRIMANTES_3D/blob/b0ba36f662e1e2850cdf81203b2ab84ad15976f0/QIDI/IFAST/CALIBRATION/Etape%2004/Z2_Calibration_Script/Z2_generate.py). I will add my new series (the most common ones) to this directory over time so you can benefit from them, but you will always need to remember to replace the **Filament Diameter #2** value with that of your own filament from [Step 03: Filament diameter calibration](https://github.com/sudtek/IMPRIMANTES_3D/blob/main/QIDI/IFAST/CALIBRATION/Etape%2003/Etape_03.md):

```
;_START_[Filament Information]
...
;Note: Updating this value with YOUR FILAMENT diameter is mandatory to find your correct flow rate!
;Filament Diameter #2: 1.724
...
;_END_[Filament Information]
```
